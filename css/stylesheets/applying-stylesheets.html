1.1. Inline styling

Code:

     <h1 style="color:purple">CSS is awesome</h1>

Output:

CSS is awesome

1.2. Internal styling

Code:

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>My website</title>
            </head>
            <style>
                h1 {
                    color: lightblue;
                }
            </style>
            <body>
                <h1>CSS Internal styling is awesome!</h1>
                
            </body>
            </html>

Output:

CSS Internal styling is awesome!

1.3. External stylesheet

Code:

index.html style.css 
        <!DOCTYPE html>
        <html lang="en">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My website</title>
        <link rel="stylesheet" href="style.css">
        </head>
        <body>
        <h1>External stylesheet is awesome!</h1>
        </body>
        </html>

                
            h1 {
                color: crimson;
            }

Output:

External stylesheet is awesome!

1.4. Cascading stylesheet

Code:

index.html style.css 
        <!DOCTYPE html>
        <html lang="en">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My website</title>
        <link rel="stylesheet" href="style.css">
        </head>
        <body>
        <h1 style="color:pink">Cascading stylesheets</h1>
        </body>
        </html>

                
            h1 {
                color: blue;
            }

Output:

Cascading stylesheets
