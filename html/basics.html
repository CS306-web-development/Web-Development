1.1. HTML documents structure

        <!doctype html>
        <html lang="en-US">
          <head>
            <meta charset="utf-8" />
            <title>My Webpage</title>
          </head>
          <body>
            <p>Hello, world</p>
          </body>
        </html>

1.2. Headers and paragraphs

Code:

        <h1>Main Header</h1>
        <h2>Subheader</h2>
        <p>This is a paragraph</p>

Output:

Main Header

Subheader

This is a paragraph

1.4. Hyperlinks

Code:

<a href="#">Click here to go nowhere</a> or 
<a href="https://www.google.com">Visit Google</a> or
<a href="https://www.google.com" target="_blank">Visit Google in a new tab</a>

Output:

Click here to go nowhere or Visit Google or Visit Google in a new tab
