Capstone Project #2: Calculator

Project Details

You are asked to implement a basic calculator using HTML, CSS and JavaScript.

Consider the following requirements:
  • Your app should have a simple calculator layout that contains buttons for numbers, buttons for operations, a button for clear and a screen for the display.
  • Your app should perform basic arithmetic operations on numbers [i.e. addition, subtraction, multiplication, division].
  • Your app should display correctly on smaller screen sizes.
  • Assume that all numbers are integers.
  • Store the previous number, current number and operation then calculate and update the display.
  • Division by 0 should display an error. [optional]
  • Implement a clear button to clear the display and reset the calculator.

You are free to use the tools of your choice, but here's some recommended tools to consider:

  • IDE : VS Code
  • Extensions for VS Code: Live Server
  • Browser : Google Chrome, Mozilla Firefox
